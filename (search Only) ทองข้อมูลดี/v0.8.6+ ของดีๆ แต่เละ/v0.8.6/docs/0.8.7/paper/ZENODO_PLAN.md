# UET Data Archiving Plan (Zenodo)

## Goal
To provide a immutable, citable record of the UET v0.8.7 validation data and code.

## Zenodo Metadata
- **Title:** Unified Energy Theory (UET) Validation Suite v0.8.7
- **Creators:** UET Research Team
- **Description:** This dataset contains the full 52-test validation suite for UET, including the stability tester, raw results JSON, and topic-specific literature reviews.
- **License:** Creative Commons Attribution 4.0 International
- **Keywords:** Physics, Unification, Scalar Field, Dark Energy, Numerical Validation

## Files to Archive
1. `src/` - Core UET harness code.
2. `docs/0.8.7/` - Full literature review and documentation.
3. `results/` - The `stability_test_report.json` and topic figures.
4. `run_all_tests.py` - The verification script.

## DOI Integration
Once the upload is complete, the DOI (e.g., `10.5281/zenodo.XXXXX`) will be cited in:
- `INDEX.md`
- `uet_paper.tex`
- GitHub `README.md`

## Schedule
- **Upload:** Upon completion of Phase 1 paper refinement.
- **Version Control:** Future versions (v0.9.0+) will be uploaded as new versions on the same DOI entry.
