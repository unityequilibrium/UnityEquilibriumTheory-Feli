# UET Wiki & Documentation Plan

## Goal
To create a searchable, user-friendly high-level overview of UET for the scientific community.

## Platform
- **MkDocs with Material Theme** (hosted on GitHub Pages).
- **Domain:** `docs.uet-theory.org` (or similar).

## Site Structure

### 1. Home
- Elevator pitch: "Force unification via energy gradients."
- Quick Start: Run the harness in 3 commands.

### 2. Foundations
- The Energy Field Equation (derivation).
- Lagrangian and Hamiltonian formalism.

### 3. The Forces
- **Gravity:** MOND behavior, M/r relationship.
- **Electromagnetism:** Fine structure constant calculation.
- **Nuclear Forces:** Yukawa potentials and mass generation.

### 4. Validation Suite
- Interactive table of all 52 tests.
- Links to raw code for each test.

### 5. Publications
- Download PDF of the arXiv paper.
- Citation instructions (BibTeX).

### 6. Contribute
- Roadmap to v1.0.0.
- How to add new physical tests.
