### 3.2 ‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡πÇ‡∏Ñ‡πâ‡∏î: ‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á Graph ‡πÉ‡∏ô LangGraph

LangGraph ‡∏™‡∏£‡πâ‡∏≤‡∏á‡πÄ‡∏ß‡∏¥‡∏£‡πå‡∏Å‡πÇ‡∏ü‡∏•‡∏ß‡πå‡πÇ‡∏î‡∏¢‡∏ô‡∏¥‡∏¢‡∏≤‡∏°‡∏™‡∏ñ‡∏≤‡∏ô‡∏∞ (State), ‡πÇ‡∏´‡∏ô‡∏î (Nodes), ‡πÅ‡∏•‡∏∞‡πÄ‡∏™‡πâ‡∏ô‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏° (Edges)

```python

from langgraph.graph import StateGraph, END

# 1. ‡∏Å‡∏≥‡∏´‡∏ô‡∏î State ‡∏Ç‡∏≠‡∏á Graph

    messages: List[BaseMessage]
    # ... other state variables like research_data, final_report

# 2. ‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡πÅ‡∏ï‡πà‡∏•‡∏∞ Node
def research_agent(state: AgentState):
    # Logic for research agent to gather information
    print("ü§ñ Research Agent: Gathering information...")
    # ... calls tools, processes data
    return {"research_data": ...}

    # Logic for analysis agent
    print("üß† Analysis Agent: Analyzing findings...")
    return {"analysis_complete": True, ...}

def report_agent(state: AgentState):
    # Logic for report generation agent
    print("üìä Report Agent: Generating comprehensive report...")
    return {"final_report": ...}

# 3. ‡∏™‡∏£‡πâ‡∏≤‡∏á Workflow Graph

# 4. ‡πÄ‡∏û‡∏¥‡πà‡∏° Nodes ‡πÄ‡∏Ç‡πâ‡∏≤‡πÑ‡∏õ‡πÉ‡∏ô Graph
workflow.add_node("research", research_agent)
workflow.add_node("analysis", analysis_agent)
workflow.add_node("report", report_agent)

# 5. ‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡πÄ‡∏™‡πâ‡∏ô‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏° (Edges) ‡πÅ‡∏•‡∏∞‡πÄ‡∏á‡∏∑‡πà‡∏≠‡∏ô‡πÑ‡∏Ç

```
